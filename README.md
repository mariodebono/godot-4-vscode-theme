# Godot 4 VS Code Theme

This is a Visual Studio Code theme inspired by the Godot 4 editor. It aims to provide a similar look and feel to the Godot editor, making it easier for developers to switch between the two environments.


## Features

This theme includes several fixes and enhancements to closely match the Godot editor's coloring:

### Color Theme

The editor colors are set as close as possible to the Godot 4 Editor

### Syntax Highlighting

Include some additional highlighting to match the script highlighting in the editor as much as possible:

- **GLobals**: includes global functions like `print`, `printt`, `print_rich`, and `printerr`.
- **Constants** includes constants like `true`, `false`, `null`, and `PI`, as well as the `void` keyword and `self` variable.
- **Comments**: includes highlighting for comment keywords such as `TODO`, `FIXME`, `NOTE`, and more.
- **Regions** - include highlighting for `#region` and `#endregion`
